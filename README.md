# 🗨️ ChatApp

A full-stack real-time chat application built with **Next.js**, **Node.js/Express**, and **MongoDB**. Users can register, log in, and send real-time messages to others.

## 🚀 Features

- 🔐 Authentication (Login/Register)
- 💬 Real-time chat with Socket.IO
- 🧾 Conversation history
- 📦 RESTful APIs (Express)
- 🌐 Responsive frontend (Next.js)
- 🗄️ MongoDB database

## 🛠️ Tech Stack

**Frontend:**
- Next.js
- React
- Axios
- Tailwind CSS

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- Socket.IO
- JSON Web Tokens (JWT)
